## Использование `return`

Ключевое слово `return` передает значение из функции обратно в точку вызова:

```python
def square(x):
    return x * x

result = square(5)
print(result)  # 25
```

Без `return` функция вернет `None`:

```python
def square_no_return(x):
    x * x

print(square_no_return(5))  # None
```

## Возврат нескольких значений

Python позволяет возвращать несколько значений через кортеж:

```python
def min_max(numbers):
    return min(numbers), max(numbers)

minimum, maximum = min_max([3, 7, 1])
print(minimum, maximum)  # 1 7
```

Можно также получить кортеж целиком:

```python
values = min_max([3, 7, 1])
print(values)  # (1, 7)
```

## Использование возвращаемых значений

Возвращаемые значения можно использовать напрямую в выражениях:

```python
def add(a, b):
    return a + b

print(add(3, 5) * 2)  # 16
```

Или передавать результат в другие функции:

```python
def multiply(a, b):
    return a * b

print(add(2, multiply(3, 4)))  # 14
```

## Частые ошибки

* Пропуск `return` → функция возвращает `None`.
* Попытка вернуть несколько значений без упаковки в кортеж → синтаксическая ошибка.
* Игнорирование возвращаемого значения → результат функции теряется.

## Часто задаваемые вопросы

1. **Можно ли вернуть список или словарь?**
   Да, любые объекты можно вернуть:

```python
def get_list():
    return [1, 2, 3]
```

2. **Можно ли вернуть результат условно?**
   Да, `return` может использоваться в любых условиях:

```python
def check(x):
    if x > 0:
        return "positive"
    return "non-positive"
```

3. **Можно ли использовать `return` в лямбда-функции?**
   Нет, лямбда-функции автоматически возвращают результат выражения, `return` не используется.

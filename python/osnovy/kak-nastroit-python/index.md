## Введение

Правильная настройка Python — важный этап перед началом разработки. Она включает установку интерпретатора, настройку среды, управление пакетами и подготовку инструментов для эффективной работы. В этой статье мы разберём, как настроить Python и подготовить рабочее окружение.

### Установка Python

Python доступен на официальном сайте [python.org](https://www.python.org/). Рекомендуется использовать последнюю стабильную версию Python 3.

Для Windows:

1. Скачайте установщик Python.
2. При установке отметьте опцию **Add Python to PATH**.
3. Следуйте инструкциям мастера установки.

Для macOS:

```bash
brew install python
```

Для Linux (Debian/Ubuntu):

```bash
sudo apt update
sudo apt install python3 python3-pip
```

### Проверка установки

Проверьте версию Python и pip:

```bash
python3 --version
pip3 --version
```

### Настройка виртуальной среды

Виртуальные среды позволяют изолировать зависимости разных проектов:

```bash
python3 -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows
```

После активации можно устанавливать пакеты локально, не влияя на глобальную систему.

### Управление пакетами с помощью pip

Для установки сторонних библиотек используется pip:

```bash
pip install requests
pip install numpy==1.25.0
```

Список установленных пакетов можно посмотреть командой:

```bash
pip list
```

### Настройка IDE и редакторов

Популярные редакторы и IDE:

* **VS Code** — лёгкая настройка, интеграция с Python.
* **PyCharm** — полноценная IDE для Python, поддержка виртуальных сред.

Настройка включает выбор интерпретатора, подключение виртуальной среды и настройку линтеров для проверки кода.

### Частые ошибки

* Неактивированная виртуальная среда
* Установка пакетов глобально вместо локально
* Несовместимые версии Python и библиотек
* Отсутствие настройки PATH на Windows

### Частозадаваемые вопросы

**Как создать виртуальную среду в Windows и Linux?**
Используйте `python -m venv venv`, затем активируйте: `venv\Scripts\activate` (Windows) или `source venv/bin/activate` (Linux/macOS).

**Как обновить pip?**

```bash
python -m pip install --upgrade pip
```

**Как проверить установленную версию Python?**

```bash
python --version
```

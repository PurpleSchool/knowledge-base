## Введение

Python — один из самых популярных языков программирования, удобный для решения как простых, так и сложных задач. Он широко используется для веб-разработки, анализа данных, автоматизации и научных вычислений. Чтобы эффективно работать с Python, важно правильно установить среду, освоить базовые команды и понимать, как запускать скрипты.
В этой статье мы разберём основные шаги работы с Python на практике.

### Установка Python

Первый шаг — установка интерпретатора Python. На официальном сайте доступны версии для Windows, macOS и Linux. Рекомендуется использовать последнюю стабильную версию Python 3.x.

```bash
# Проверка установленной версии Python
python --version
# или для некоторых систем
python3 --version
```

После установки можно перейти к настройке среды разработки.

### Настройка среды разработки

Для работы с Python удобно использовать текстовые редакторы или IDE. Популярные варианты:

* **VS Code** — легкая и гибкая среда с поддержкой Python и плагинов.
* **PyCharm** — мощная IDE с интеграцией отладки и инструментов для тестирования.
* **IDLE** — базовая среда, поставляемая вместе с Python, подходит для быстрых экспериментов.

Для управления проектами и зависимостями рекомендуется создавать виртуальные окружения:

```bash
# Создание виртуального окружения
python -m venv venv
# Активация виртуального окружения (Windows)
venv\Scripts\activate
# Активация виртуального окружения (macOS/Linux)
source venv/bin/activate
```

Виртуальное окружение позволяет изолировать зависимости одного проекта от других.

### Запуск скриптов Python

Python поддерживает запуск скриптов напрямую из консоли:

```bash
python script.py
# или
python3 script.py
```

Также можно запускать интерактивную консоль для экспериментов:

```bash
python
# или
python3
```

### Основные команды

* **print()** — вывод информации на экран.
* **input()** — получение данных от пользователя.
* **type()** — проверка типа данных.
* **len()** — длина строки, списка или другой коллекции.

```python
name = input("Введите имя: ")
print(f"Привет, {name}!")
print(type(name))
```

### Частые ошибки

* Использование Python 2 вместо Python 3.
* Отсутствие активации виртуального окружения.
* Ошибки при указании пути к скрипту.
* Пропуск обязательных символов, например, двоеточия `:` в конструкциях `if`, `for`, `def`.

### Частозадаваемые вопросы

**Как проверить версию Python?**
`python --version` или `python3 --version`.

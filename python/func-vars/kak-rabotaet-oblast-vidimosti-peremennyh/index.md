## Типы области видимости

Python использует правило **LEGB** для поиска переменных:

* **L (Local)** — локальная область функции.
* **E (Enclosing)** — область внешней функции, если используется вложенная функция.
* **G (Global)** — глобальная область модуля.
* **B (Built-in)** — встроенные переменные Python (`len`, `range` и др.).

### Пример локальной области

```python
def func():
    x = 10  # локальная переменная
    print(x)

func()  # 10
# print(x)  # Ошибка, x не определена в глобальной области
```

### Пример глобальной области

```python
y = 20  # глобальная переменная

def func():
    print(y)

func()  # 20
```

### Вложенные функции и Enclosing

```python
def outer():
    a = 5
    def inner():
        print(a)  # доступ к enclosing области
    inner()

outer()  # 5
```

## Изменение глобальных переменных

Для изменения глобальной переменной внутри функции используется `global`:

```python
counter = 0

def increment():
    global counter
    counter += 1

increment()
print(counter)  # 1
```

## Частые ошибки

* Попытка изменить глобальную переменную без `global`.
* Использование локальных переменных до их определения.
* Перезапись встроенных функций Python (`len`, `sum` и др.).

## Часто задаваемые вопросы

1. **Можно ли обращаться к переменной из внешней функции без передачи аргументов?**
   Да, вложенные функции могут использовать переменные enclosing области.

2. **Что делать, если нужно сохранить значение между вызовами функции?**
   Используйте глобальные переменные, mutable объекты, или замыкания.

3. **Можно ли изменить встроенную функцию?**
   Технически можно, но это крайне не рекомендуется, так как нарушает поведение языка.

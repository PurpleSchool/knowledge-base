## Введение

Консоль Python — это интерактивная среда, которая позволяет быстро тестировать код, выполнять вычисления и отлаживать скрипты без необходимости создавать полноценный файл. Владение консолью ускоряет разработку и помогает лучше понимать работу языка.
В этой статье мы разберём, как пользоваться консолью Python на практике.

### Запуск консоли

Для запуска интерактивной оболочки Python выполните команду:

```bash
python
```

или, если у вас установлено несколько версий:

```bash
python3
```

Вы увидите приглашение `>>>`, которое сигнализирует о готовности к вводу команд.

### Основные команды

В консоли можно выполнять любые выражения Python:

```python
>>> 2 + 3
5
>>> print("Hello, Python")
Hello, Python
>>> x = 10
>>> x * 2
20
```

Консоль поддерживает работу с переменными, функциями и импорт библиотек:

```python
>>> import math
>>> math.sqrt(16)
4.0
```

Для выхода из консоли используйте:

```python
>>> exit()
```

или комбинацию `Ctrl + D`.

### Работа с скриптами

Консоль позволяет запускать готовые файлы с кодом:

```bash
python my_script.py
```

Все команды из файла выполняются последовательно, а результат выводится на экран. Это удобно для тестирования и отладки небольших программ.

### История команд и автодополнение

Интерактивная оболочка запоминает предыдущие команды — можно использовать стрелки `↑` и `↓` для навигации. Также доступно автодополнение при помощи клавиши `Tab`, что ускоряет ввод длинных имен переменных и функций.

### Частые ошибки

* Попытка использовать переменные до их объявления.
* Ошибки синтаксиса при копировании кода в консоль.
* Неправильный путь к скрипту при запуске файла.
* Забытие импорта необходимых модулей.

### Частозадаваемые вопросы

**Как выйти из консоли Python?**
Через команду `exit()` или комбинацию `Ctrl + D`.

**Можно ли запускать консоль с конкретной версией Python?**
Да, используйте `python3` или полный путь к нужной версии.

**Как сохранить результат работы консоли?**
Можно перенаправить вывод в файл или использовать скрипт вместо интерактивного режима.

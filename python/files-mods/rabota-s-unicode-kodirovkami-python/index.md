## –í–≤–µ–¥–µ–Ω–∏–µ

Unicode —Å—Ç–∞–ª —Å—Ç–∞–Ω–¥–∞—Ä—Ç–æ–º –¥–ª—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è —Å–∏–º–≤–æ–ª–æ–≤ –∏–∑ —Ä–∞–∑–Ω—ã—Ö —è–∑—ã–∫–æ–≤ –∏ —Å–∏—Å—Ç–µ–º –ø–∏—Å—å–º–∞, –∏ Python –ø–æ–ª–Ω–æ—Å—Ç—å—é –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç –µ–≥–æ –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å–æ —Å—Ç—Ä–æ–∫–∞–º–∏. –ü–æ–Ω–∏–º–∞–Ω–∏–µ Unicode –≤–∞–∂–Ω–æ –ø—Ä–∏ –æ–±—Ä–∞–±–æ—Ç–∫–µ —Ç–µ–∫—Å—Ç–∞, —á—Ç–µ–Ω–∏–∏ –∏ –∑–∞–ø–∏—Å–∏ —Ñ–∞–π–ª–æ–≤, –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏–∏ —Å API –∏ —Å–µ—Ç–µ–≤—ã–º–∏ —Å–µ—Ä–≤–∏—Å–∞–º–∏. –í —ç—Ç–æ–π —Å—Ç–∞—Ç—å–µ –º—ã —Ä–∞–∑–±–µ—Ä—ë–º, –∫–∞–∫ —Ä–∞–±–æ—Ç–∞—Ç—å —Å Unicode –∫–æ–¥–∏—Ä–æ–≤–∫–∞–º–∏ –≤ Python –∏ –∏–∑–±–µ–∂–∞—Ç—å —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω—ë–Ω–Ω—ã—Ö –æ—à–∏–±–æ–∫ –ø—Ä–∏ –æ–±—Ä–∞–±–æ—Ç–∫–µ —Ç–µ–∫—Å—Ç–æ–≤—ã—Ö –¥–∞–Ω–Ω—ã—Ö.

### –°—Ç—Ä–æ–∫–∏ –∏ Unicode –≤ Python

–í Python 3 –≤—Å–µ —Å—Ç—Ä–æ–∫–∏ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω—ã –≤ —Ñ–æ—Ä–º–∞—Ç–µ Unicode:

```python
text = "–ü—Ä–∏–≤–µ—Ç, –º–∏—Ä üåç"
print(text)
print(len(text))  # 13, –∫–∞–∂–¥—ã–π —Å–∏–º–≤–æ–ª —É—á–∏—Ç—ã–≤–∞–µ—Ç—Å—è –∫–∞–∫ –æ–¥–∏–Ω
```

Unicode –ø–æ–∑–≤–æ–ª—è–µ—Ç —Ä–∞–±–æ—Ç–∞—Ç—å —Å —Å–∏–º–≤–æ–ª–∞–º–∏ —Ä–∞–∑–ª–∏—á–Ω—ã—Ö —è–∑—ã–∫–æ–≤, —ç–º–æ–¥–∑–∏ –∏ —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã—Ö –∑–Ω–∞–∫–æ–≤ –±–µ–∑ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç–∏ —è–≤–Ω–æ–≥–æ —É–∫–∞–∑–∞–Ω–∏—è –∫–æ–¥–∏—Ä–æ–≤–∫–∏ –¥–ª—è —Å—Ç—Ä–æ–∫ –≤ –ø–∞–º—è—Ç–∏.

–î–ª—è —Å–∏—Å—Ç–µ–º–Ω–æ–≥–æ –æ—Å–≤–æ–µ–Ω–∏—è —Ä–∞–±–æ—Ç—ã —Å Unicode, —Ç–µ–∫—Å—Ç–æ–≤—ã–º–∏ –¥–∞–Ω–Ω—ã–º–∏ –∏ –±–µ–∑–æ–ø–∞—Å–Ω–æ–π –æ–±—Ä–∞–±–æ—Ç–∫–æ–π —Ñ–∞–π–ª–æ–≤ –ø–æ–ª–µ–∑–Ω–æ –∏–∑—É—á–∏—Ç—å –∫—É—Ä—Å [–û—Å–Ω–æ–≤—ã Python](https://purpleschool.ru/course/python-basics?utm_source=knowledgebase&utm_medium=article&utm_campaign=Rabota_s_Unicode_kodirovkami_Python). –ö—É—Ä—Å –≤–∫–ª—é—á–∞–µ—Ç 209 —É—Ä–æ–∫–æ–≤, 34 —É–ø—Ä–∞–∂–Ω–µ–Ω–∏—è, AI-—Ç—Ä–µ–Ω–∞–∂—ë—Ä—ã –¥–ª—è –ø—Ä–∞–∫—Ç–∏–∫–∏ 24/7, –∂–∏–≤–æ–µ —Ä–µ–≤—å—é –Ω–∞—Å—Ç–∞–≤–Ω–∏–∫–æ–≤ –∏ –µ–∂–µ–Ω–µ–¥–µ–ª—å–Ω—ã–µ –≤—Å—Ç—Ä–µ—á–∏, —á—Ç–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç –∑–∞–∫—Ä–µ–ø–∏—Ç—å –Ω–∞–≤—ã–∫–∏ —Ä–∞–±–æ—Ç—ã —Å —Ç–µ–∫—Å—Ç–æ–º –≤ —Ä–µ–∞–ª—å–Ω—ã—Ö –ø—Ä–æ–µ–∫—Ç–∞—Ö.

### –ü—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ —Å—Ç—Ä–æ–∫ –∏ –±–∞–π—Ç–æ–≤

–î–ª—è –ø–µ—Ä–µ–¥–∞—á–∏ –¥–∞–Ω–Ω—ã—Ö –ø–æ —Å–µ—Ç–∏ –∏–ª–∏ –∑–∞–ø–∏—Å–∏ –≤ —Ñ–∞–π–ª —Å—Ç—Ä–æ–∫–∏ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞—Ç—å –≤ –±–∞–π—Ç—ã:

```python
s = "–ü—Ä–∏–≤–µ—Ç"
b = s.encode("utf-8")  # str ‚Üí bytes
print(b)  # b'\xd0\x9f\xd1\x80\xd0\xb8\xd0\xb2\xd0\xb5\xd1\x82'

decoded = b.decode("utf-8")  # bytes ‚Üí str
print(decoded)  # –ü—Ä–∏–≤–µ—Ç
```

–ü—Ä–∏ —Ä–∞–±–æ—Ç–µ —Å —Ä–∞–∑–Ω—ã–º–∏ –∏—Å—Ç–æ—á–Ω–∏–∫–∞–º–∏ –¥–∞–Ω–Ω—ã—Ö –≤–∞–∂–Ω–æ –ø—Ä–∞–≤–∏–ª—å–Ω–æ —É–∫–∞–∑—ã–≤–∞—Ç—å –∫–æ–¥–∏—Ä–æ–≤–∫—É –ø—Ä–∏ –∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏–∏ –∏ –¥–µ–∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏–∏.

### –ß—Ç–µ–Ω–∏–µ –∏ –∑–∞–ø–∏—Å—å —Ñ–∞–π–ª–æ–≤ —Å Unicode

Python –ø–æ–∑–≤–æ–ª—è–µ—Ç –±–µ–∑–æ–ø–∞—Å–Ω–æ —Ä–∞–±–æ—Ç–∞—Ç—å —Å —Ñ–∞–π–ª–∞–º–∏ —Å —É–∫–∞–∑–∞–Ω–∏–µ–º –∫–æ–¥–∏—Ä–æ–≤–∫–∏:

```python
# –ó–∞–ø–∏—Å—å —Ñ–∞–π–ª–∞ —Å UTF-8

    f.write("–¢–µ—Å—Ç —Å —ç–º–æ–¥–∑–∏ üåü")

# –ß—Ç–µ–Ω–∏–µ —Ñ–∞–π–ª–∞
```

–≠—Ç–æ –≥–∞—Ä–∞–Ω—Ç–∏—Ä—É–µ—Ç –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ–µ –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏–µ —Å–∏–º–≤–æ–ª–æ–≤ –Ω–∞ –ª—é–±—ã—Ö —Å–∏—Å—Ç–µ–º–∞—Ö –∏ –ø—Ä–∏ –ø–æ—Å–ª–µ–¥—É—é—â–µ–π –æ–±—Ä–∞–±–æ—Ç–∫–µ.

### –ß–∞—Å—Ç—ã–µ –æ—à–∏–±–∫–∏

* –ü–æ–ø—ã—Ç–∫–∞ —á–∏—Ç–∞—Ç—å —Ñ–∞–π–ª —Å –∫–æ–¥–∏—Ä–æ–≤–∫–æ–π, –æ—Ç–ª–∏—á–Ω–æ–π –æ—Ç –∏—Å–ø–æ–ª—å–∑—É–µ–º–æ–π (`UnicodeDecodeError`).
* –°–º–µ—à–∏–≤–∞–Ω–∏–µ –±–∞–π—Ç–æ–≤ –∏ —Å—Ç—Ä–æ–∫ –±–µ–∑ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è (`TypeError`).
* –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ Python 2, –≥–¥–µ —Å—Ç—Ä–æ–∫–∏ –Ω–µ —è–≤–ª—è—é—Ç—Å—è Unicode –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é.

### –ß–∞—Å—Ç–æ–∑–∞–¥–∞–≤–∞–µ–º—ã–µ –≤–æ–ø—Ä–æ—Å—ã

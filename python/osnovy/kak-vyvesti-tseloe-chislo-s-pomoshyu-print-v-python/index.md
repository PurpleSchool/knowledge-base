## Введение

Вывод информации на экран — базовая задача при работе с Python. Особенно важно правильно выводить числа, чтобы результаты вычислений или данные переменных отображались корректно. В этой статье мы разберём, как вывести целое число с помощью функции `print` и использовать дополнительные возможности форматирования.
В этой статье мы разберемся.

### Вывод целого числа

Функция `print()` позволяет отображать целые числа и другие типы данных:

```python
x = 42
print(x)
```

Результат будет просто числом `42`.

Также можно выводить несколько значений одновременно:

```python
a = 10
b = 5
print("Сумма:", a + b)
```

Функция объединяет текст и вычисленные значения.

### Форматирование вывода

Вывод чисел можно улучшить с помощью форматирования строк:

```python
age = 25
print(f"Возраст пользователя: {age}")
```

или с использованием метода `format`:

```python
score = 100
print("Счет игрока: {}".format(score))
```

Это позволяет комбинировать текст и числа более гибко и удобно.

### Преобразование типов при выводе

Если нужно вывести число вместе со строкой без форматирования, требуется преобразовать его в строку:

```python
level = 7
print("Уровень: " + str(level))
```

Без `str()` Python выдаст ошибку `TypeError`.

### Частые ошибки

* Попытка объединить число и строку без преобразования типов (`TypeError`).
* Использование устаревшего синтаксиса Python 2 (`print x` вместо `print(x)`).
* Ошибки в формате f-строк при неправильном использовании фигурных скобок.

### Частозадаваемые вопросы

**Можно ли выводить несколько чисел одновременно?**
Да, через запятую: `print(a, b, a+b)`.

**Как вывести число с текстом без пробелов между ними?**
Использовать `end=""` или метод `str()`: `print("Число: " + str(x))`.

**Можно ли выводить числа в разных системах счисления?**
Да, через функции `bin()`, `oct()`, `hex()`.

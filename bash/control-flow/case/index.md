## Синтаксис

```bash
case "$command" in
  start) echo "Запуск" ;;
  stop) echo "Остановка" ;;
  *) echo "Неизвестная команда"; exit 1 ;;
esac
```

## Итоги

`case` делает ветвление более читаемым и устойчивым для CLI-сценариев.

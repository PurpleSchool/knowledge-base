## Создание окна

Начнём с базового окна приложения:

```python
import tkinter as tk

root = tk.Tk()
root.title("Моё приложение")
root.geometry("300x200")

root.mainloop()
```

* `Tk()` создаёт главное окно.
* `title` задаёт название окна.
* `geometry` определяет размер окна.
* `mainloop()` запускает цикл обработки событий.

## Добавление кнопок и меток

Элементы интерфейса создаются через виджеты:

```python
label = tk.Label(root, text="Привет, Python GUI!")
label.pack()

def on_click():
    label.config(text="Вы нажали кнопку")

button = tk.Button(root, text="Нажми меня", command=on_click)
button.pack()
```

* `Label` отображает текст.
* `Button` реагирует на нажатие через `command`.
* `pack()` автоматически размещает элементы в окне.

## Поля ввода и обработка данных

Добавим поле ввода и обработку текста:

```python
entry = tk.Entry(root)
entry.pack()

def show_entry():
    user_text = entry.get()
    label.config(text=f"Вы ввели: {user_text}")

button_entry = tk.Button(root, text="Показать текст", command=show_entry)
button_entry.pack()
```

`Entry` позволяет вводить данные пользователю, а `get()` возвращает текст для обработки.

## Компоновка элементов

Tkinter предлагает несколько способов размещения виджетов: `pack()`, `grid()`, `place()`. Пример с сеткой:

```python
tk.Label(root, text="Имя:").grid(row=0, column=0)
tk.Entry(root).grid(row=0, column=1)
tk.Button(root, text="Отправить").grid(row=1, columnspan=2)
```

`grid()` позволяет точно управлять положением элементов по строкам и колонкам.

## Работа с событиями и обратными вызовами

Можно привязывать действия к событиям, например, к нажатию клавиши:

```python
def key_pressed(event):
    label.config(text=f"Нажата клавиша: {event.char}")

root.bind("<Key>", key_pressed)
```

`bind` связывает событие с функцией-обработчиком.

## Частые ошибки

* Не запуск `mainloop()` — окно не отображается.
* Использование `pack()` и `grid()` одновременно в одном контейнере.
* Неправильное присвоение функции в `command` (например, `command=on_click()` вместо `command=on_click`).

## Часто задаваемые вопросы

1. **Можно ли менять размер окна динамически?**
   Да, через `root.geometry()` или методы `resizable()`.

2. **Как обрабатывать несколько кнопок и полей ввода?**
   Создайте отдельные функции для каждого действия или используйте `lambda`.

3. **Можно ли использовать другие библиотеки GUI с Tkinter?**
   Да, например, `ttk` для стильных виджетов или `PIL` для работы с изображениями.

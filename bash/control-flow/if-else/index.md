## Базовая форма if

```bash
if [[ условие ]]; then
  echo "ok"
else
  echo "fail"
fi
```

## Полезные проверки

```bash
[[ -z "${TOKEN:-}" ]]
[[ -f "$file" ]]
[[ -d "$dir" ]]
```

## Числовые сравнения

```bash
if (( retries > 3 )); then
  echo "Слишком много попыток"
fi
```

## Итоги

Используй `[[ ... ]]`, покрывай ветку ошибок и не забывай кавычки.

## local

`local` ограничивает переменную рамками функции.

```bash
sum() {
  local a="$1"
  local b="$2"
  echo $((a+b))
}
```

## return vs exit

- `return` — завершает функцию;
- `exit` — завершает весь скрипт.

```bash
validate() {
  [[ -n "${1:-}" ]] || return 1
  return 0
}
```

## Итоги

Используй `local` в функциях по умолчанию и возвращай статус через `return`, а не через глобальные переменные.

## Объявление массива

```bash
services=(api worker scheduler)
```

## Доступ к элементам

```bash
echo "${services[0]}"
echo "${services[@]}"
echo "${#services[@]}"
```

## Безопасный обход

```bash
for svc in "${services[@]}"; do
  echo "$svc"
done
```

## Добавление и удаление

```bash
services+=(gateway)
unset 'services[1]'
```

## Итоги

Массивы полезны для пакетной обработки. Основное правило — обходить элементы через `"${arr[@]}"`.
